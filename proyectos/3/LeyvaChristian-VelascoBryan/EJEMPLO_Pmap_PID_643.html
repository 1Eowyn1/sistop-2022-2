  • Comprendiste el uso de la página.

Stack

  • Este uso es el de las funciones y variables locales del sistema.

Biblioteca

  • En esta paginación se esta accediendo a las bibliotecas del sistema ubicadas en el directorio /lib.

Heap

  • Segmento de memoria que no tiene un tamaño constante antes de la compilación, el cual es controlado dinámicamente por el programador.

Llamada al sistema

  • vDSO significa "virtual dynamic shared object" la cual es una pequeña biblioteca que el kernel mapea automaticamente en el espacio compartido de aplicaciones.

  • Muchas llamadas al sistema que proporciona el kernel utilizan el código de espacio de usuario por lo que las llamadas al sistema pueden dominar el rendimiento general.

Procesos Var (Kernel)

  • Con el objetivo de implementar algunas llamadas al sistema en especifico, el kernel mapea algunos datos para el usuario.

Permisos de archivos

  • En UNIX los permisos dentro de los archivos se representan con 3 Letras:

  • r: read -> lectura

  • w: write -> escritura (en archivos: permiso de modificar, en directorios: permiso de crear archivos en el dir.)

  • x execution -> ejecución
