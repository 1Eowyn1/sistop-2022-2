Instrucciones

El programa que analice fue un pequeño programa que tenía realizado en el lenguaje Python el cual simplemente muestra un temporizador en la consola de ejecucion del tiempo que el usuario le ingresa al inicio del mismo, asi como la hora en que el contador inicio y el contador termino.
El codigo fuente mostrado a continuacion contiene marcado de color ROJO las instrucciones que yo considero que realizan llamadas al sistema, al dar click en ellas te dirigira a la explicacion de porque yo considero que realiza una llamada al sistema.

Codigo



    import time
    from datetime import datetime
    from os import system
    from os import name as OSName

    def clear():
        # nt -> Windows
        if OSName == 'nt':
            system('cls')
        else:
            system('clear')

    def contador(tiempo_seg):
        # Obtiene la hora actual
        hora_inicio = datetime.now() 

        while tiempo_seg:
            mins, segs = divmod(tiempo_seg, 60)
            tiempo_str = '{:02d}:{:02d}'.format(mins, segs)
            # end \r -> Escribe en la misma linea.
            print(tiempo_str, end='\r') 
            time.sleep(1)
            tiempo_seg -= 1
        hora_final = datetime.now()
        
        print("¡Tiempo agotado!")
        print("Hora inicio: {}".format(hora_inicio.strftime("%H:%M:%S")))
        print("Hora termino: {}\n".format(hora_final.strftime("%H:%M:%S")))

    def main():
    print("Ingresa el tiempo del contador EN SEGUNDOS: ", end= '')
        tiempo = int(input())
        clear()
        mins, segs = divmod(tiempo, 60)
        tiempo_str = '{:02d}:{:02d}'.format(mins, segs)
        print("Contador de {} min.".format(tiempo_str))
        contador(tiempo)

    main()
